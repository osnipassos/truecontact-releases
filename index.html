Os seus dados são seus

O TrueContact funciona completamente offline. A base de dados SQLite fica no seu computador — nenhum contato, foto ou nota é enviado para servidores externos. A única conexão com a internet é para sincronizar com o seu iCloud e Google Contacts, verificar atualizações no LinkedIn e enviar mensagens pelo WhatsApp.

Pronto para começar?

Sem conta. Sem subscrição. Os seus dados ficam consigo.
