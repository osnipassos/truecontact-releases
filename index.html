Os seus dados são seus

O TrueContact funciona completamente offline. A base de dados SQLite fica no seu computador — nenhum contacto, foto ou nota é enviado para servidores externos. A única ligação à internet é para sincronizar com o seu iCloud e verificar atualizações.

Gratuito. Sem conta. Sem subscrição. Os seus dados ficam consigo.
