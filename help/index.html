Secção 1

Primeiros Passos

O que é o TrueContact?

O TrueContact é um CRM pessoal automático para Mac e Windows. Liga-se ao iCloud, ao Google Contacts, ao LinkedIn e ao WhatsApp para sincronizar, enriquecer e organizar os teus contatos — e envia mensagens de aniversário automaticamente, em segundo plano.

Ao contrário das soluções cloud (HubSpot, Salesforce, etc.), o TrueContact corre 100% no teu computador. Não há subscrição, não há conta, não há servidor a salvar os seus dados.

Para quem é:

  • Profissionais de networking que querem manter contatos sempre atualizados
  • Quem usa iPhone/Mac e quer tirar o máximo partido do iCloud Contacts
  • Quem usa o Gmail e quer manter o Google Contacts em sincronia com o iCloud
  • Quem quer enviar parabéns automáticos sem parecer um bot
  • Quem valoriza a privacidade total dos seus dados de contato
Requisitos do sistema e como instalar

macOS

Ventura 13 ou superior

Windows

Windows 10 (64-bit) ou superior

Passos de instalação:

  1. Acesse a página de download e baixe o arquivo para o seu sistema
  2. macOS: Abre o .dmg e arrasta o TrueContact para Aplicações
  3. Windows: Executa o instalador .exe e segue os passos
  4. Na primeira abertura, o Assistente de Configuração guia-te por todas as integrações
macOS: Se aparecer o aviso "app de desenvolvedor desconhecido", vai a Configurações do Sistema → Privacidade e Segurança e clica em "Abrir na mesma". O TrueContact não está na App Store por opção — isso é intencional para manter a app gratuita.
Qual é a sequência de configuração recomendada?

Na primeira abertura, o Assistente de Configuração guia-te automaticamente por todos estes passos:

  1. 1 Configurar o iCloud — introduz o Apple ID e a App-Specific Password para importar os teus contatos
  2. 2 Conectar o Google Contacts — autoriza via OAuth para sincronização bidirecional com o Google
  3. 3 Conectar o WhatsApp — lê o QR Code nas Configurações para ativar o envio de mensagens
  4. 4 Criar templates de aniversário — define as mensagens automáticas que o sistema vai enviar
  5. 5 Deixar correr — o TrueContact faz tudo o resto em segundo plano

Secção 2

Segurança e Privacidade

Onde ficam armazenados os meus dados?

100% no teu próprio computador. O TrueContact usa uma base de dados SQLite local — nenhum contato, foto, nota ou histórico de mensagens é enviado para servidores externos. Não existem servidores TrueContact.

macOS

~/Library/Application Support/truecontact/database.sqlite

Windows

%APPDATA%\truecontact\database.sqlite

Podes abrir, copiar ou fazer backup deste ficheiro com ferramentas como o DB Browser for SQLite.

O TrueContact envia algum dado para a internet?

O TrueContact apenas faz conexões diretamente com os serviços que já usas — nada passa por servidores TrueContact:

iCloud CardDAV

Conexão direta aos servidores da Apple — as tuas credenciais nunca passam pelo TrueContact

Google People API

Autenticação via OAuth 2.0 — o token fica local e nunca passa por servidores TrueContact

LinkedIn

Lê perfis públicos como um browser normal — só leitura, nenhum dado teu é enviado

WhatsApp Web

Conexão direta aos servidores do WhatsApp — sessão autenticada localmente

GitHub Releases

Apenas verifica um número de versão — nenhum dado pessoal envolvido

Como funciona a sessão do WhatsApp — é segura?

Quando ligas o WhatsApp através do QR Code, é gerado um ficheiro de sessão local no teu computador. Este ficheiro mantém a autenticação sem necessidade de introduzir credenciais repetidamente.

macOS

~/Library/Application Support/truecontact/whatsapp-session/

Windows

%APPDATA%\truecontact\whatsapp-session\
Aviso: Nunca compartilhes esta pasta com ninguém. Quem tiver acesso a estes ficheiros pode enviar mensagens em teu nome. Se suspeitas de comprometimento, vai a WhatsApp → Configurações → Dispositivos conectados e remove a sessão TrueContact.
O que é a App-Specific Password do iCloud?

O TrueContact usa uma App-Specific Password para aceder ao iCloud Contacts — nunca a tua password principal do Apple ID. É uma password de uso único gerada pela Apple especificamente para esta app.

Vantagens:

  • Se for comprometida, podes revogá-la sem alterar o teu Apple ID
  • Dá acesso apenas ao CardDAV (Contatos), não ao restante do iCloud
  • A Apple permite revogar individualmente por app

Para gerar, vai a appleid.apple.com → Iniciar sessão e segurança → Senhas específicas de app.

Secção 3

Sincronização de Contatos

Como o TrueContact une dados do iCloud, LinkedIn e WhatsApp?

O TrueContact funciona como um Hub central: a base de dados local é a fonte de verdade, e os provedores bidirecionais (iCloud, Google Contacts) lêem e escrevem contatos, enquanto LinkedIn e WhatsApp apenas enriquecem.

  1. 1 Pull do iCloud — lê todos os contatos e grupos via CardDAV, preservando ETags para detetar alterações
  2. 2 Pull do Google — lê alterações desde o último sync via SyncToken, bidirecional
  3. 3 Enriquecimento LinkedIn — importa cargo, empresa, foto e redes sociais automaticamente
  4. 4 Enriquecimento WhatsApp — verifica a foto de perfil de cada contato (se autorizado)
  5. 5 Push para iCloud e Google — escreve as alterações de volta em ambos, ficando disponíveis em todos os dispositivos
Regra fundamental: O TrueContact nunca apaga dados manuais. Se adicionares uma nota ou email a um contato no iPhone, essa informação é preservada mesmo que o LinkedIn devolva um campo vazio.
Como funciona a normalização de números de telefone?

O mesmo número pode estar guardado de formas completamente diferentes no iCloud e no WhatsApp. O TrueContact normaliza-os automaticamente para fazer a correspondência:

(11) 98765-4321 → normalizado → 5511987654321
+55 11 9 8765-4321 → normalizado → 5511987654321
11 8765-4321 → + 9º dígito → 5511987654321

Inclui suporte ao 9º dígito brasileiro para números de DDDs onde a transição ocorreu depois de 2012 — sem que precises de atualizar os teus contatos manualmente.

Porque não vejo a foto de alguns contatos do WhatsApp?

Esta é uma das perguntas mais frequentes. A resposta é simples: o WhatsApp respeita as configurações de privacidade de cada utilizador.

Se um contato configurou a sua foto de perfil para ser visível apenas para os seus próprios contatos (ou para ninguém), o WhatsApp não disponibiliza essa foto a nenhuma aplicação — incluindo o TrueContact. Quando isso acontece, o TrueContact mostra um ícone de privacidade em vez da foto, sinalizando que o contato existe no WhatsApp mas optou por restringir a sua imagem.

Isto é um comportamento correto, não um bug. O TrueContact respeita as escolhas de privacidade dos teus contatos da mesma forma que o próprio WhatsApp faz.
Dica: Se quiseres ver a foto de um contato específico, pede-lhe que abra WhatsApp → Configurações → Privacidade → Foto de perfil → "Todos os Meus Contatos".
Como o TrueContact deteta e resolve contatos duplicados?

O TrueContact analisa automaticamente a tua base de dados em busca de contatos com nome, email, telefone ou perfil social semelhantes.

Quando encontra sugestões:

  • Na sidebar, aparece o bloco Sugestões de Merge
  • Clica em Rever para comparar os dois contatos lado a lado
  • Escolhe quais os campos a manter de cada lado

Para fazer merge manual:

  1. Passa o cursor sobre um contato → aparece um ✓ no avatar
  2. Seleciona 2 ou mais contatos
  3. Clica em Merge na barra inferior

Secção 4

Conectar o Google Contacts

Passo a passo: autorização OAuth
  1. Abra o Assistente de Configuração (na primeira abertura) ou vá a Configurações → Google Contacts
  2. Clique em "Conectar Google Contacts"
  3. Uma janela de autenticação do Google será aberta no seu navegador
  4. Faça login com a sua conta Google e conceda as permissões solicitadas
  5. O TrueContact receberá o token automaticamente e iniciará a sincronização inicial
O TrueContact usa OAuth 2.0 — você nunca insere a sua senha do Google no app. O token fica armazenado localmente e pode ser revogado a qualquer momento em myaccount.google.com/permissions.
O que acontece na primeira conexão?

Quando você conecta o Google Contacts pela primeira vez:

  1. 1. O TrueContact lê todos os contatos existentes no Google
  2. 2. Todos os contatos locais que ainda não estão no Google são enfileirados para criação (propagação total automática)
  3. 3. A sincronização ocorre gradualmente para respeitar os limites da API Google
  4. 4. Um badge azul Google no cabeçalho indica quantos contatos ainda estão a ser processados
Grupos: Os grupos de contatos também são sincronizados. Um grupo criado no iCloud aparece automaticamente no Google Contacts, e vice-versa.
Como revogar o acesso ao Google?
  1. Vá a myaccount.google.com/permissions
  2. Localize "TrueContact" na lista
  3. Clique em "Revogar acesso"

Isso não apaga os contatos do seu Google Contacts — apenas interrompe a sincronização futura.

Secção 5

Conectar o WhatsApp

Passo a passo: como ler o QR Code

No TrueContact:

  1. 1. Abre as Configurações (ícone ⚙️)
  2. 2. Vai à secção WhatsApp
  3. 3. Clica em "Conectar WhatsApp"
  4. 4. Aparece um QR Code — tens 60 segundos

No teu celular:

  1. 1. Abre o WhatsApp
  2. 2. Vai a Definições → Dispositivos conectados
  3. 3. Toca em "Conectar um dispositivo"
  4. 4. Aponta a câmara para o QR Code
O TrueContact aparece como "Chrome em [nome do computador]" na lista de dispositivos conectados. O WhatsApp permite até 4 dispositivos ao mesmo tempo.
Resolver problemas de conexão

O QR Code expira antes de conseguir ler

Clica em "Atualizar QR Code" nas Configurações. O código é regenerado instantaneamente.

"Sessão desligada" aparece nas Configurações

O WhatsApp desliga dispositivos inativos por mais de 14 dias. Repete o processo de conexão com o QR Code.

O WhatsApp diz "Número de aparelhos atingido"

Remove um dispositivo existente em WhatsApp → Configurações → Dispositivos conectados e tenta novamente.

As mensagens não estão a ser enviadas

Verifica em Configurações → WhatsApp se o estado mostra "Conectado". Se não, reconecta através do QR Code.

Secção 6

Templates e Automações

Como criar templates de mensagem de aniversário?
  1. Abre as Configurações
  2. Vai à secção Templates de Aniversário
  3. Clica em "Adicionar Template"
  4. Escreve a mensagem, usando {{nome_amigavel}} para o nome do contato
  5. Clica em "Salvar"

Exemplos de templates:

Olá {{nome_amigavel}}! 🎂 Hoje é o teu dia especial — muitas felicidades e que este novo ano traga tudo o que mereces! 🥳
{{nome_amigavel}}, parabéns pelo teu dia! 🎉 Que seja um ano cheio de conquistas e momentos especiais.
Feliz aniversário, {{nome_amigavel}}! 🎈 Que a vida te sorria sempre.
Cria 3 a 6 variações diferentes. O sistema garante que não envia a mesma mensagem duas vezes seguidas ao mesmo contato.
O que é a variável {{nome_amigavel}} e como funciona?

A variável {{nome_amigavel}} é substituída automaticamente pelo nome mais natural do contato. O sistema segue esta ordem de prioridade:

1

Apelido (Nickname)

Se o contato tiver um apelido definido no iCloud (ex: Zé), usa esse

2

Primeiro nome

Se não tiver apelido, usa o primeiro nome (ex: José)

3

Nome completo

Apenas como último recurso

Exemplo prático:

O teu contato está guardado como "José Manuel Ferreira da Silva" no iCloud, mas toda a gente o chama de "Zé". Com o apelido definido, a mensagem será:

"Olá Zé! 🎂..."

em vez de: "Olá José Manuel Ferreira da Silva! 🎂..."

Dica: Para adicionar um apelido no iPhone, abre o contato → "Editar" → "adicionar campo" → "Apelido".
Como funcionam o sorteio de mensagens e o delay de digitação?

Sorteio Aleatório

Quando há múltiplos templates, o sistema escolhe um aleatoriamente. O mesmo contato receberá uma mensagem diferente a cada aniversário.

Delay de Digitação

Antes de enviar, ativa o indicador "a escrever..." e aguarda um tempo proporcional ao comprimento da mensagem — como se fosses tu a escrever.

Aviso: Não configures templates com linguagem de marketing ou links promocionais. O WhatsApp pode suspender contas que enviam mensagens em massa com conteúdo publicitário. Os templates devem ser mensagens genuínas e pessoais.

Secção 7

Atualizações

Como funciona a atualização automática?

O TrueContact usa um sistema de atualização OTA (Over-The-Air) — silencioso, automático e seguro. Não precisas de fazer nada.

  1. 1 Quando a app inicia, verifica discretamente se existe uma versão mais recente
  2. 2 Se encontrar uma versão nova, baixa em segundo plano sem interromper o teu trabalho
  3. 3 Uma notificação discreta pergunta se queres instalar agora ou na próxima abertura
  4. 4 A atualização é instalada e a app reinicia automaticamente
As atualizações são seguras?

Sim. Cada ficheiro de atualização é assinado criptograficamente antes de ser publicado. O TrueContact verifica esta assinatura antes de instalar qualquer atualização — se a assinatura não corresponder, a atualização é rejeitada automaticamente. Isto protege-te contra ficheiros corrompidos ou adulterados.

Nota técnica: As atualizações são publicadas no repositório público osnipassos/truecontact-releases. O código-fonte nunca é publicado nesse repositório — apenas os binários compilados e assinados.

Podes sempre ver a versão instalada em Configurações → Sobre o TrueContact, no fundo da página.
