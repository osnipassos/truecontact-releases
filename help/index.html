Ajuda & FAQ

Respostas às perguntas mais frequentes sobre o TrueContact.

Privacidade e Segurança

Os meus dados ficam guardados onde?

Os teus dados ficam 100% no teu próprio computador. O TrueContact usa uma base de dados SQLite local — nenhum contacto, foto ou nota é enviado para servidores externos.

A única ligação à internet ocorre para:

  • Sincronizar com o teu iCloud Contacts (usando as tuas próprias credenciais Apple)
  • Verificar atualizações da aplicação
  • Abrir perfis do LinkedIn no browser (se usares o radar de networking)

Sincronização com o iCloud

Como ligo o TrueContact ao iCloud Contacts?
  1. Abre as Configurações (ícone ⚙️ no canto superior direito)
  2. Na secção Apple / iCloud, introduz o teu Apple ID e uma App-Specific Password
  3. Para gerar uma App-Specific Password, vai a appleid.apple.com → Segurança → Palavras-passe específicas de app
  4. Clica Guardar — o TrueContact vai importar os teus contactos automaticamente

Os teus contactos são mantidos em sincronização bidirecional: alterações feitas no iPhone ou Mac aparecem no TrueContact, e vice-versa.

Aniversários

Como funcionam os envios automáticos de parabéns?

O TrueContact verifica diariamente os aniversários dos teus contactos e pode enviar uma mensagem de parabéns de forma automática.

Para configurar:

  1. Vai a Configurações → Aniversários
  2. Ativa a opção Enviar parabéns automaticamente
  3. Escolhe o template de mensagem ou cria o teu próprio
  4. Define a hora do envio (por omissão: 09:00)

As mensagens são enviadas via WhatsApp Web ou pelo canal que configurares. O histórico de envios aparece no card Datas Importantes de cada contacto.

O sistema usa o ano 1604 internamente para aniversários sem ano definido (ex: importados do LinkedIn). Isto é intencional e não afeta os envios.

Duplicados e Merge

Como o TrueContact deteta e resolve contactos duplicados?

O TrueContact analisa automaticamente a tua base de dados em busca de contactos com nome, email, telefone ou perfil social semelhantes.

Quando encontra sugestões:

  • Aparece uma notificação no sistema
  • Na sidebar, aparece o bloco Sugestões de Merge
  • Podes clicar em Rever para comparar os dois contactos lado a lado

Para fazer merge manual:

  1. Na lista de contactos, passa o cursor sobre um contacto → aparece um ✓ no avatar
  2. Seleciona 2 ou mais contactos
  3. Clica no botão Merge na barra inferior
